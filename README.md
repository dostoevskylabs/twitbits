# twitbits
stupid nodejs twitter bot to store data in a database individual to each user messaging it.

Essentially it works like this to add an item
```code
.a #kali #cmd #hash hash-identifier
.a #readlater https://bit.ly/somelink
```

Find items
```code
.s #kali #cmds
.s #readlater
```

List available tags
```code
.t
```

Delete entry
```code
.d id
```

Display help
```code
.h
```
